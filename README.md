# 🎬 Video Downloader

A simple Python application for downloading videos with a clean interface built using Flet.

---

## 🚀 Features

* 📥 Download videos from YouTubes
* ⚡ Fast and lightweight
* 🖥️ Simple and user-friendly interface
* 🎨 Theme switching (light/dark mode)

---

## 📦 Installation

### 1. Clone the repository

```
git clone https://github.com/your_username/video-downloader.git
cd video-downloader-YT
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Install ffmpeg

This project requires ffmpeg.

You can use it in two ways:

Install ffmpeg globally and add it to your system PATH

Or simply download ffmpeg.exe and place it in the project folder

---

## ▶️ Usage

Run the application:

```
python UI.py
```

---
